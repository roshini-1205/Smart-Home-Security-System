# Smart Home Security System

## Objective
To create a simple home security system using Arduino Uno, LCD Display, Push Button and Buzzer.

## Components Used
- Arduino Uno
- 16x2 LCD Display
- Push Button
- Active Buzzer
- Potentiometer
- Breadboard
- Jumper Wires

## Working
1. LCD displays "Home Secure".
2. Push button acts as a door sensor.
3. When the button is pressed:
   - LCD displays "ALERT! Intruder"
   - Buzzer sounds.

## Applications
- Home Security
- Door Monitoring
- Educational Projects
